# Apprentice+

Your Course, Your Way

An apprenticeship tracking and management application built with React and TypeScript.

## Features

- **Homepage**: Track your apprenticeship progress
- **Academy**: Access short in-house style training sessions
- **Apprenticeship**: View KSB Matrix and track assignments
- **Documents**: Manage course-related documents
- **Settings**: Configure your course selection

## Getting Started

```bash
npm install
npm start
```

## Tech Stack

- React 18
- TypeScript
- Material-UI
- React Router
- Zustand (State Management)

## Color Scheme

- Primary: #7FCC39 (Lime Green)
- Secondary: #1B3A52 (Dark Teal)
- Accent: #FFFFFF (White)
